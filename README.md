# Template_React_ExpressAPI_Docker_MySQL
Un template React avec une API Express lié à une BD local Docker MySQL.
## Installation
### Général
Créer la base de données docker
```
docker compose up
```
### API
```
npm install
npm start
```
### monApp
```
npm install
npx vite
```
